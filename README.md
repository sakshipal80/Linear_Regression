# 🎓 Student Performance Prediction (Linear Regression)

## 📌 Overview

This project predicts students' final marks based on key factors such as:

* Study Hours
* Sleep Hours
* Phone Usage
* Attendance

It uses a **Linear Regression model** to analyze how these variables impact academic performance.

---

## ⚙️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 📊 Features

* Predict student marks using Machine Learning
* Multi-feature input (study, sleep, phone, attendance)
* Model training and evaluation
* Performance metrics (MAE, R² Score)
* Data visualization

---

## 🧠 Model Used

* Linear Regression

---

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* R² Score
